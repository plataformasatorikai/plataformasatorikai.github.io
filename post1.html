Introducción a Python

Python es un lenguaje de programación interpretado de alto nivel que se utiliza en una variedad de aplicaciones. Algunos de sus usos comunes incluyen la automatización de tareas, la ciencia de datos y el desarrollo de aplicaciones web. Para empezar en Python, necesitas tener instalado el intérprete de Python en tu ordenador. Puedes descargarlo desde el sitio web oficial.

Una vez que lo tienes instalado, puedes abrir el intérprete de Python y comenzar a escribir código. Primero, podemos imprimir un mensaje simple en la pantalla:

print("Hola, mundo!")

Este código imprimirá "Hola, mundo!" en la pantalla. También podemos hacer algunas operaciones matemáticas simples:

print(2 + 2)

Este código imprimirá "4" en la pantalla. Finalmente, podemos utilizar variables para almacenar información y hacer operaciones con ella:

x = 5 y = 10 print(x + y)

Este código imprimirá "15" en la pantalla. ¡Y eso es una introducción básica a Python! Espero que te haya gustado este pequeño tutorial.
